# World Food Map 🍴🌍

전 세계 180개국 이상의 전통 음식을 탐색할 수 있는 인터랙티브 지도 애플리케이션입니다.

## 🚀 주요 기능
- **180+ 국가 데이터**: 각국의 대표 음식 상세 설명 제공.
- **실시간 이미지**: Wikimedia Commons API를 통한 음식 사진 표시.
- **인터랙티브 맵**: 6가지 색상으로 구분된 고해상도 확대/축소 지도.

## 🛠️ 기술 스택
- React, Vite, Bootstrap 5
- react-simple-maps, Lucide React

## 📦 시작하기
1. **의존성 설치**
   ```bash
   npm install
   ```
2. **실행**
   ```bash
   npm run dev
   ```
